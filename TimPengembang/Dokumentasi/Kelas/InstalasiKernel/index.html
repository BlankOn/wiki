Instalasi Kernel

melanjutkan dari halaman ini, maka langkah selanjutnya adalah:

1.buat file /etc/kernel-img.conf (di chroot), isi dengan: 

do_bootloader = No
do_initrd = Yes 
warn_initrd = No

2.Masih di chroot, silahkan install kernelnya:
