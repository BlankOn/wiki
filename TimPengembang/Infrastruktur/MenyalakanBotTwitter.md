# Menyalakan Bor Twitter'
Bot Twitter ini bertujuan untuk rettwet bagi akun twitter yang ngetwit dengan
#BlankOn dan dikuti oleh @BlankOnLinux?
Caranya Masuk Ke Waljinah
$ sudo su - situs
$ screen -S botblankon
$ cd botblankon
$ python botblankon.py
